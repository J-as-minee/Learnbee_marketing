# Course: Learnbee 101 — what it is, and how to build your first course

**Audience:** New course creators evaluating or onboarding to Learnbee (L&D teams, trainers)
**Objective:** Learner can explain what Learnbee does and build a first draft course end to end
**Language:** English · **Default voice:** Female
**Modules:** 4 · **Slides:** 18 · **Quizzes:** 4 (1 per module)

Each slide below is written as: **format** (a real Learnbee slide format ID) → on-screen content → narration script. Narration for quiz/true-false slides is a single short instruction line, per Learnbee's own narration convention — the question text itself isn't read aloud.

## Module 2 — Building a course with the AI Wizard

**2.1 · agenda-slide**
- Heading: "In this module"
- Items: Basics · Content · Media · Structure · Quiz · Review
- Narration: "Every course starts in the same place — a six-step wizard. Here's what those six steps cover."

**2.2 · step-by-step** *(click mode — one segment per step)*
1. **Basics** — Title, audience, objective, language, and default voice. Narration: "Step one: tell Learnbee the basics — your title, audience, objective, language, and a default voice. The language locks in here, so choose carefully."
2. **Content** — Paste reference text, upload a PDF, or turn on web search. Narration: "Step two: give Learnbee something to work from — paste in reference text, upload a PDF, or let it search the web."
3. **Media** — Drag in images, videos, and course PDFs. Narration: "Step three: drop in any images, videos, or course PDFs you want included — they're tagged to this course automatically."
4. **Structure** — Set module count, duration, and depth. Narration: "Step four: decide how big the course should be — how many modules, how long, and how deep to go."
5. **Quiz** — Set quiz count, difficulty, and distribution. Narration: "Step five: set how many quiz questions you want, how hard they should be, and how they're spread across the course."
6. **Review** — Check everything, then hit Generate. Narration: "Step six: review everything one last time, then hit Generate — Learnbee builds your full draft course from here."

**2.3 · insight-cards**
- Card 1: "Language is locked" — Pick it carefully in step one; it can't be changed later.
- Card 2: "Reference text does the heavy lifting" — The more specific your input, the better the draft.
- Card 3: "You can skip media" — Add images and videos later from the editor if you don't have them yet.
- Narration: "Three things worth knowing before you start the wizard."

**2.4 · quiz-mcq**
- Question: "Which wizard step is where you set how many quiz questions the course will have?"
- Options: A) Basics  B) Structure  C) Quiz  D) Review
- Correct answer: C) Quiz
- Narration: "Quick check — pick your answer."

## Module 4 — Publishing & sharing

**4.1 · comparison-pros-cons**
- "SCORM export" — Pros: works in any LMS, fully self-contained, trackable scores. Cons: takes a 30–40 second build step.
- "Publish & share" — Pros: live in seconds, shareable link + QR + embed code. Cons: no LMS gradebook.
- Narration: "Two ways to ship a finished course — here's how they compare."

**4.2 · step-by-step** *(click mode)*
1. **Export or publish** — Choose from the editor. Narration: "From the editor, choose Export for an LMS, or Publish for a shareable link."
2. **SCORM packages itself** — Images and audio bundle in automatically. Narration: "Exporting packages everything — images and audio included — into one self-contained zip."
3. **Publish gives you a code instantly** — Narration: "Publishing gives you a six-character access code the moment it's done."
4. **Share it your way** — Code, link, QR, or embed. Narration: "Share that code, the link, the QR code, or the embed snippet — whichever fits."
5. **Republish anytime** — Same code keeps working. Narration: "Made an edit? Republish — the same code and link keep working, no need to reshare."

**4.3 · big-statement**
- Statement: "Your first course can be live today."
- Narration: "Start from a document, a topic, or a blank page — your first course can be live today."

**4.4 · quiz-mcq**
- Question: "What do you get when you publish a course instead of exporting it to SCORM?"
- Options: A) A graded LMS transcript  B) A 6-character access code, link, QR, and embed code  C) A PDF download  D) A new course wizard
- Correct answer: B
- Narration: "Last question — pick your answer."
